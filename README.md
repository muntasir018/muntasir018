## 🚀 What I've Built

### 🛡️ [DeskWarden](https://github.com/muntasir018/DeskWarden)
> A Windows application locker — password-protect any `.exe` on your PC, just like your phone.

- Built with Python + PyQt6 from scratch
- Kernel-level process suspension via Windows API
- Custom one-click installer with real-time progress bars
- Modern dark UI — designed to feel like paid software

### 🪵 KathGhor (কাঠঘর)
> A full-featured timber shop management PWA — everything a wood 
> business needs, in one place.

- Bilingual (Bengali + English) — fully switching UI
- CFT/piece pricing with per-type rate management
- Invoice generation, PDF billing, and QR sharing
- Cart system, order history, and customer tracking
- Built-in tools that turn hours of manual work into seconds
- Firebase Firestore sync + offline-first with localStorage fallback
- Google Auth + guest mode with isolated data
- Single-file PWA — no framework, no build step, installs like an app

### 🤖 Project Mina
> An autonomous Windows AI agent that thinks and acts on its own — 
> no hardcoded shortcuts, no predefined rules.

- Understands natural language commands and executes them intelligently
- Scans and controls installed apps, browsers, and system functions
- Web search with multi-source fallback chain
- Voice control with real-time speech recognition
- Makes its own decisions — powered by Cohere command-r-plus
